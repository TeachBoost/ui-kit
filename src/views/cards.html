Information Cards

Color cards present quick, high-level data to users on their Dashboard and profile pages. These come in a number of colors and should typically contain a number/statistic; the Dashboard version doesn't contain an icon and is larger, the profile version is tighter and always contains an icon. They may be clickable and provide a link below them to click through to more details or a fuller page with tabular data, charts, and the like.

Cards can and should float left in a horizontal line from each other (they carry a default display: inline-block; so make sure to override this with a different display if necessary. Their markup uses the <figure> and <figcaption> elements.

Big Cards

Big cards show either one number or figure, or a large icon. Their caption id displayed in a dark overlay at the top of the card and generally always links to a table or breakdown of the data they represent. These cards are only displayed on the Dashboard currently.

To create a big card use a <figure> with classes .big .card. Place the card's caption inside of <figcaption> with class .caption and place the card's value inside either a link or span (if it's not clickable) with class .value. These cards carry an 18% width so that five fit nicely on one line. Add a color class to color it, otherwise they will display in Ash. 

<!-- Big Dashboard cards -->
<figure class="big teal card">
    <figcaption class="caption">
        Total
    </figcaption>
    <span class="value">
        37k
    </span>
</figure>
<figure class="big blue card">
    <figcaption class="caption">
        Complete
    </figcaption>
    <a href="javascript:;" class="value">
        2453
    </a>
</figure>
...

Little Cards

Little cards are used on Profile pages and carry a value/figure and a representational icon to the left. These cards may also be either static (use a <span>) or clickable with a link. A caption is optional and if included will appear below the card. Note that for links, the caption text automatically adds in the → to the right—no need to manually add that little guy! 

<!-- Static card with no caption -->
<figure class="blue icon card">
    <span class="value">
        <i class="fa fa-users"></i> 123
    </span>
</figure>

<!-- Static card with a caption -->
<figure class="green icon card">
    <span class="value">
        <i class="fa fa-trophy"></i> 45
    </span>
    <figcaption class="caption">
        Span with text
    </figcaption>
</figure>

<!-- Clickable card with no caption (discouraged) -->
<figure class="purple icon card">
    <a href="javascript:;" class="value">
        <i class="fa fa-eye"></i> 1.7
    </a>
</figure>

<!-- Clickable card with a caption -->
<figure class="red icon card">
    <a href="javascript:;" class="value">
        <i class="fa fa-book"></i> 110
    </a>
    <figcaption class="caption">
        <a href="javascript:;">Link with text</a>
    </figcaption>
</figure>
