Icons

TeachBoost uses Font Awesome for it font-based application icons. Please refer to Font Awesome's full icon list for the classnames for each icon. You can find our list of icons selected by function below; always check there to ensure a consistent user experience and brand.

<i> Is for Icon

Font Awesome uses the <i> element to display icons. There are only two classes you need to add to it: the .fa base class and the icon symbol's class .fa-*. The library's examples page shows all of the available classnames for added functionality but we only use the size amplifier among these (see below for more). Always remember to close the tag or else you'll get wonky looking text! 

<!-- Basic icon representing our dashboard -->
<i class="fa fa-tachometer"></i>

<!-- Calendar icon that is double sized -->
<i class="fa fa-calendar fa-2x"></i>

Icon Sizes

Icons default at the size of the text they sit in, which is typically 15px. To change the size of the icon to two, three, four, or five times the size use the following classes respectively: .fa-2x, .fa-3x, .fa-4x, and .fa-5x.
